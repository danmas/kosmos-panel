## What I do

Я выполняю комплексную проверку OpenSSH сервера:

1. Проверяю установлен ли OpenSSH сервер
2. Проверяю статус службы ssh/sshd
3. Проверяю прослушивает ли сервер порт 22
4. Тестирую локальное подключение через ssh localhost
5. Показываю последние записи в логах SSH сервера
6. Проверяю конфигурационный файл на наличие очевидных ошибок

Я всегда показываю вывод всех команд и даю рекомендации по устранению проблем.

## When to use me

Используй меня, когда:
- Нужно проверить работает ли SSH сервер на машине 
- Возникают проблемы с подключением по SSH
- Требуется диагностика неисправностей SSH сервиса
- Нужно убедиться, что SSH сервер корректно установлен и настроен
- Проверяешь безопасность системы и открытые сервисы
